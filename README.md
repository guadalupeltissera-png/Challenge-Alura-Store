# Challenge - Alura Store

## Descripción del proyecto

Este proyecto analiza el desempeño de 4 tiendas pertenecientes a Alura Store con el objetivo de ayudar al Sr. Juan a decidir cual tienda debería vender.

El análisis se basa en datos de ventas, rendimiento y reseñas de clientes, buscando identificar la tienda menos eficiente a través de métricas cuantitativas y visualizaciones claras.

## Objetivo

Evaluar el desempeño de cada tienda analizando:
  * Ingresos totales
  * Categorías más vendidas
  * Productos más vendidos
  * Reseñas de clientes
  * Promedio de envío
  * Rendimiento general de ventas

Y por ultimo, con toda esta información poder emitir una recomendación fundamentada sobre cual tienda debería venderse.

## Herramientas Utilizadas

  * Pandas
  * Matplotlib
  * Google Colab
  * GitHub
